Atom code editor

  • Windows, Linux and MacOS compatible
  • Pretty much supports everything with enough hacking and extensions
  • FOSS under MIT license
  • Install from website
  • Can create macros and have several plugins for IntelliSense, SpellChecking, auto compiling and more.
